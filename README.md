# 双极板堆叠有限元分析：Abaqus实验脚本

本仓库保存双极板堆叠压力均匀性研究中实验01至34的建模、排序、求解和后处理脚本。项目使用实测或统计生成的双极板表面形貌，通过Abaqus分析不同堆叠顺序、压力、材料特性、约束方式和片数对层间接触压力分布的影响。

仓库只保存脚本和说明文档，不包含ODB、INP、曲面CSV、图片及求解日志等生成文件。

## 研究目标

比较以下三种堆叠方式：

- 最小距离和：尽量让相邻双极板的表面形貌接近。
- 自然顺序：按照双极板原始编号堆叠。
- 最大距离和：尽量让相邻双极板的表面形貌差异最大。

主要假设是：

```text
压力均匀性：最小距离和 > 自然顺序 > 最大距离和
```

这里的“更好”表示压力离散程度、峰值、Gini系数和高载荷集中程度更低，同时承压节点覆盖率更高。实验结果必须通过能量、接触状态和网格质量检查，不能仅因为排序符合假设就判定有效。

## 当前结论

实验28是当前15片实测曲面摩擦接触基线，实验30是目前完整排序趋势最清晰的24片参数实验，实验34用于比较旧序列与Held-Karp精确DP序列。

- 最小距离和排序在CV、P95/mean、峰均比、Gini、Top20载荷占比、接触覆盖率等指标上稳定优于另外两种排序。
- 自然顺序与最大距离和的差异较小，部分归一化指标方向不一致。
- 实验30在CV、P95/mean、Gini、Top20载荷占比和覆盖率上得到“最小优于自然、自然优于最大”的完整顺序。
- 实验34的精确DP序列也恢复了主要整体指标的严格顺序，但精确min没有超过实验28旧min，说明几何距离全局最优不等于CPRESS全局最优。
- 实验29和30已经运行；实验31至33仍为待运行参数实验。

## 实验29-33快速运行

进入需要运行的实验目录：

```powershell
cd "29-15片低压高刚度角点定位三排序Abaqus参数实验"
```

### 1. 生成曲面、排序和INP

```powershell
.\run_prepare_only.ps1
```

这一步会：

1. 读取原始15片曲面。
2. 按配置复制或生成指定数量的曲面。
3. 计算最小距离和、自然顺序和最大距离和。
4. 写出三个Abaqus输入文件。

该脚本不会启动Abaqus求解。

### 2. 提交Abaqus作业

运行三种排序：

```powershell
.\run_abaqus_jobs.ps1
```

只运行一种排序：

```powershell
.\run_abaqus_jobs.ps1 -Cases min
.\run_abaqus_jobs.ps1 -Cases natural
.\run_abaqus_jobs.ps1 -Cases max
```

### 3. 后处理

将三种排序的ODB放回同一个`outputs`目录，然后运行：

```powershell
.\run_postprocess.ps1
```

后处理输出包括节点接触压力、能量历史、三种排序汇总指标和对比图。

## 合成曲面与排序

实验30至33从原始15片实测曲面生成更多板面：

- 保留原始201 x 85网格和物理尺寸。
- 在原始形貌之间插值。
- 叠加固定随机种子的低频平滑扰动和小幅局部噪声。
- 板号中保留缓慢制造漂移，同时加入可复现随机偏移。

自然顺序因此具有中等相邻差异；最小距离算法可进一步连接相似板面；最大距离算法倾向于交替连接差异较大的板面。该设计只影响几何样本，不预设Abaqus压力结果。

距离定义沿用原项目：对应基准点之间的欧氏距离之和。大规模排序使用多起点贪心路径，并通过2-opt继续优化。

## 结果有效性判据

至少检查以下项目：

1. Abaqus任务以`COMPLETED SUCCESSFULLY`结束。
2. 恒压末段动能与内能之比`KE/IE`足够低，通常目标小于5%。
3. 人工能量与内能之比`AE/IE`优先控制在10%以内。
4. 载荷、反力与目标名义压力相符。
5. 内部界面接触压力均被成功提取。
6. 承压覆盖率不存在明显非物理稀疏化。
7. 壳单元没有严重翘曲、负体积或异常穿透。
8. 三种排序除堆叠顺序外采用完全相同的参数。

均匀性比较至少同时报告：

- 变异系数CV
- P95/mean
- peak/mean
- Gini系数
- Top20载荷占比
- 承压节点覆盖率
- 绝对压力峰值
- 相对目标压力的RMSE
